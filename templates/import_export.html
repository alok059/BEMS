🔍 Smart Column Mapping

The system automatically recognizes these column name variations. No need to match exact names!

Equipment Name: name, equipment name, device name, title
Model No: model, model no, model number, model #
Company: company, manufacturer, brand, vendor, make
Serial No: serial, serial no, serial number, sn, s/n
Quantity: quantity, qty, number, count, units
Department: department, dept, location, unit, ward
Status: status, condition, state, working status
Install Date: install date, installation date, date installed
Warranty Date: warranty, warranty date, warranty expiry
Last Service: last service, last maintenance, service date
✨
Tip: The system will find the right columns automatically. Only the "Name" column is required. All other columns are optional.

📌 Column Descriptions

Name - Required. Equipment name
Model No - Optional. Model number
Company - Optional. Manufacturer
Serial No - Optional. Unique identifier
Quantity - Optional. Default 1 for bulk items
Department - Optional. Auto-creates new departments
Status - Optional. Working/Under Repair/Scrap
Install Date - Optional. YYYY-MM-DD format
Warranty Date - Optional. YYYY-MM-DD format
Last Service - Optional. YYYY-MM-DD format

💡 Import Tips

✅ Only Name is Required

All other columns are optional. Missing fields will be left blank.

🔄 Auto-create Departments

New departments are automatically created from the Department column.

📦 Bulk Items

Use Quantity > 1 for identical items like oxygen regulators.

⚠️ Error Handling

Bad data in one row won't stop other rows from importing.

📅 Date Format

Use YYYY-MM-DD format (e.g., 2024-01-15) for dates.
